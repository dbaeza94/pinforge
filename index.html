Privacy Policy

PinForge - Last updated: June 2026

1. Overview

PinForge is a personal content management tool used exclusively by its developer to manage Pinterest and Instagram accounts owned by that individual. This application is not available to the public.

2. Data Collection

PinForge does not collect, store, or process personal data from any third party. The application interacts solely with the Pinterest API and Instagram Graph API on behalf of the account owner.

3. Data Storage

API credentials are stored locally on the developers own device. No data is transmitted to any third-party server operated by PinForge.

4. Third-Party Services

This application uses: Pinterest API v5 and Meta Graph API, each subject to their own privacy policies.

5. No Data Sharing

No personal data or API response data is shared with any third party by PinForge.
