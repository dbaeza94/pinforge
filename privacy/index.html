01

Overview

PinForge is a personal content management tool developed and operated exclusively by its creator to manage Pinterest and Instagram accounts that are owned by that individual.

This application is not publicly available, has no registered end-users, and does not provide any service to third parties. All interactions with external platforms are performed solely on behalf of the account owner.

02

Data Collection

PinForge does not collect, process, or store personal data from any third party. No analytics, telemetry, crash reporting, or tracking mechanisms are embedded in the application.

API responses received from Pinterest and Instagram (such as pin metrics, post insights, or account analytics) are used exclusively in real-time to display information to the account owner and are not persisted on any remote server.

03

Data Storage

API credentials (access tokens, user IDs, and API keys) are stored exclusively in two locations, both under the sole control of the account owner:

Local filesystem (.claude/settings.json)
Browser localStorage (dashboard UI only)

No credentials or user data are transmitted to, or stored on, any server operated by PinForge.

04

Third-Party Services

PinForge communicates with the following external APIs on behalf of the account owner. Each service is subject to its own independent privacy policy.

Pinterest API v5
Meta Graph API v22 (Instagram)
Nano Banana Image Generation API

PinForge does not control, and is not responsible for, the data practices of these third-party platforms.

05

No Data Sharing

PinForge does not sell, rent, disclose, or share any data — personal or otherwise — with any third party, advertiser, analytics provider, or data broker.

There are no embedded advertising SDKs, no user profiling systems, and no cross-app tracking mechanisms of any kind within this application.
